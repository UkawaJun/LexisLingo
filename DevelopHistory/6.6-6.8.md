## 2026-05-31

### 用户单词数据持久化（UserWordStoreSQLite）
- 实现 SQLite 单表存储 + 加密的单词数据持久层
- 存储模型 `UserWordF(word TEXT PK, data TEXT)`，data 为 JSON 序列化后经 EndeCrypt 加密的字符串
- 核心方法：`read_all` / `write_all` / `get` / `get_or_create` / `set` / `get_many` / `has`
- 5 万词环境下随机点查 < 0.04ms/次

### 词卡 UI 改造
- 词卡尺寸加大至 720×960
- 中间释义区加入 `QScrollArea`，内容溢出时可上下滚动
- 支持 Ctrl + 鼠标滚轮缩放字体（范围 0.6x ~ 2.5x）
- 文字默认左上对齐，缩小时靠 padding 自然居中

### 循环播放
- 词卡右上角新增 🔁 循环切换按钮（`setCheckable`）
- `_LoopPlayWorker` 子线程跑循环播放，避免阻塞 UI
- 每播完一遍发 `playedOnce` 信号交主线程处理
- 关闭词卡（X / 我知道了 / Esc）均能安全停止循环

### 暴露度追踪（increaseExpose）
- 新增权重表 `SLEL`：点击词卡 0.2 / 单次主动听 1.0 / 循环每遍 0.02
- 每次累加用 `round(x, 2)` 防浮点误差
- 函数只改传入 dict，持久化交由调用方负责

### 点读模式（🍎 按钮）
- 学习页右上角新增 🍎 点读切换按钮
- 开启后点单词直接播放音频，不弹词卡
- 借 `WordCardDialog` 实例（不 `.show()`）处理音频，5 秒后 `deleteLater()`
- 用 `_active_count` 类计数器 + `destroyed` 信号验证资源回收

### 字段编码与模板扩充
- 单词字段采用 2 字符短码（如 `exposeTimes` → `ET`），通过常量映射访问
- `wordPorp` 模板扩充：WrongTimes / LastWrong / FirstSeenAt / IsMastered / WrongHistory / Important
- `FILE_LABEL` 新增 Use1 / Use2 / CourseText / Tree / wordFrequent 路径

### Bug 修复
- 修复 SQLite 二进制文件被当 UTF-8 文本读取导致的 `UnicodeDecodeError`
- 修复词卡模板被引用清空导致的 `KeyError: TS`（删除多余 `info.clear()`，加 `is` 守卫）
- 修复暴露度 lost update 覆盖问题
- 修复浮点累加出现长尾小数（0.6000000000000001 类）
- 修复字典遍历误取 key 而非 value
- 修复点"我知道了"后循环音频不停的问题（覆写 `done()` 统一关闭路径）
