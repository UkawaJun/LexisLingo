## 2026-05-31

### 用户单词数据持久化
- 实现 `UserWordStoreSQLite`：SQLite 单表存储 + 加密
- 存储模型 `UserWordF(word PK, data)`，data 为 JSON 序列化后加密的字符串
- 核心方法：`read_all` / `write_all` / `get` / `get_or_create` / `set` / `get_many` / `has`

### 词卡 UI 改造
- 词卡尺寸加大至 720×960
- 中间释义区加入 `QScrollArea`，内容溢出可上下滚动
- 支持 Ctrl + 鼠标滚轮缩放字体（0.6x ~ 2.5x）

### 循环播放
- 词卡右上角新增 🔁 循环切换按钮
- 用 `_LoopPlayWorker` 子线程跑循环播放，不阻塞 UI
- 关闭词卡（X / 我知道了 / Esc）均能安全停止循环

### 暴露度追踪
- 新增 `increaseExpose` 与权重表 `SLEL`（点击 0.2 / 主动听 1.0 / 循环每遍 0.02）
- 每次累加用 `round(x, 2)` 防浮点误差

### 点读模式
- 学习页右上角新增 🍎 点读切换按钮
- 开启后点单词直接播放音频，不弹词卡
- 用 `_active_count` 类计数器 + `destroyed` 信号验证资源回收

### Bug 修复
- 修复 SQLite 二进制文件被当 UTF-8 文本读取导致的 `UnicodeDecodeError`
- 修复词卡模板被引用清空导致的 `KeyError: TS`
- 修复暴露度 lost update 覆盖问题
- 修复浮点累加出现长尾小数
- 修复字典遍历误取 key 而非 value
- 修复点"我知道了"后循环音频不停的问题（覆写 `done()` 统一关闭路径）

## 2026-06-01

### 自由句子合成（SentenceDialog）
- 学习页右上角新增 🍊 按钮，弹出非模态句子合成窗口
- 输入任意英文句子，支持下载与播放
- 用独立的 `_SentencePlayWorker` 子线程播放

### 句子多音色 / 多速度
- 支持 4 档音色（英女 Sonia / 英男 Ryan / 美女 Aria / 美男 Guy）
- 支持 5 档语速（115% / 100% / 85% / 70% / 55%）
- 音色与速度组合进缓存 key（格式 `[Ryan@85] 句子`）

### 音频存储分层
- 单词音频与句子音频路径、索引完全独立
- `SmartAudioAtlas` 重构为可配置路径（`sound_dir` / `index_path` / `max_entries`），向后兼容

### LRU 缓存系统
- 单词缓存上限 5000，句子上限 500
- 新增 `_touch` / `_evict` / `_enforce_max` 三个方法
- 索引写入用临时文件 + 原子替换，加线程锁，损坏文件降级为空索引

### UserConfig 重构
- 新增嵌套 `Setting` 子字典存放系统偏好
- 新增字段：`MaxWordSound` / `MaxSentSound` / `DailyGoalWords` / `DailyGoalMinutes` / `FontScale` / `TargetLang`
- 新增 `LastLoginAt` / `CurrentCourse` / `Nickname` / `Avatar` / `_v` 版本号
- 删除冗余字段

### Bug 修复
- 修复 `_Enviro` 反向 import `Config` 导致的循环引用
- 修复长句作文件名超过 Windows 路径长度限制（超 80 字符转哈希命名）

- 多余字段只读保留，支持 Ctrl+滚轮缩放

- 单词数据库可视化调试器
- 支持搜索、过滤、列排序、双击编辑全字段、右键删除
- 内存副本 + 手动保存，支持放弃修改

### 调试工具 · DebugWordKey.py
- 单词字段批量增删工具
- 批量给所有单词增加 / 删除指定字段，操作前预览影响范围
- 保存时自动备份原数据库为 `.db.copy`
- 双击单词只读查看全部字段
